# Adaptive Emotion-Aware Virtual Interaction System

## Overview
This repository contains the initial project skeleton and configuration for an adaptive emotion-aware virtual interaction system. The training pipeline is model-agnostic and can swap model backbones using shared configuration.

## Implemented Models
- MN-Xception
- EfficientNet-B0

## Planned Models
- ResNet-18
- HSEmotion

## Project Structure
- `README.md`: Project overview and current model status.
- `.gitignore`: Common Python and OS ignores.
- `requirements.txt`: Python dependencies for training and tooling.
- `config.py`: Model-agnostic configuration for all supported architectures.
